
## Instalação

#### Setup configuration:
<code> 

composer install

</code>

#### Generate application key:
<code>

cp .env.example .env

</code>

####
##### exemplo escrito dentro no BD condicionar para esta configuração
<code>

DB_CONNECTION=mysql
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=olw-1dep
DB_USERNAME=sail
DB_PASSWORD=password

</code>

#### VITE - não funciona (colocar no vite.config.js)
<code>

server: {
        hmr: {
            host: "localhost"
        }
    },


</code>

#### colocar no package.json
<code>

 "dev": "vite --host localhost", 

</code>


#### INSTALAR O NPM
<code>

npm install

</code>
